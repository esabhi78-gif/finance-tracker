# 💰 Finance Tracker

A simple, elegant web-based finance tracker to manage your income and expenses.

## Features

- ✅ **Dashboard** - View total balance, income, and expenses at a glance
- ✅ **Add Transactions** - Record income and expenses with categories, dates, and descriptions
- ✅ **Real-time Calculations** - Automatic balance updates
- ✅ **Filter & Search** - View all transactions or filter by type (income/expense)
- ✅ **Delete Transactions** - Remove individual items
- ✅ **Clear All** - Delete all transactions at once
- ✅ **Local Storage** - All data persists in your browser
- ✅ **Responsive Design** - Works on mobile, tablet, and desktop
- ✅ **Beautiful UI** - Modern gradient design with smooth animations

## Getting Started

1. Download or clone this repository
2. Open `index.html` in your web browser
3. Start adding transactions!

## How to Use

1. **Fill the form** with transaction details:
   - Description (e.g., "Coffee", "Salary")
   - Amount in dollars
   - Type (Income or Expense)
   - Category (automatically updates based on type)
   - Date

2. **Click "Add Transaction"** to save

3. **View your dashboard** for instant totals:
   - Total Balance
   - Total Income
   - Total Expenses

4. **Filter transactions** using the buttons (All, Income, Expense)

5. **Delete transactions** individually or clear all at once

## Data Storage

All transactions are saved locally in your browser using Local Storage. Your data is:
- **Private** - Never sent to any server
- **Persistent** - Remains when you close and reopen the browser
- **Easy to Export** - Check browser DevTools → Application → Local Storage
